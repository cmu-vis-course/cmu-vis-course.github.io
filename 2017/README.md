### CMU VIS Course

The 2017 Visualization website is built with [Jekyll](http://jekyllrb.com).

#### Setup

Requires Jekyll 2.4.0 or later.

```ShellSession
$ gem install jekyll
```

#### Generate, Serve and Deploy Site

```ShellSession
$ bundle exec jekyll serve --watch
$ rake generate
$ rake deploy
```

#### View Site

```ShellSession
$ open http://0.0.0.0:4000/
```
